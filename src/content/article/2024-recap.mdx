## ソフトウェア開発

日々の生活で必要になったツールをいくつか作りました。公開しているものをいくつか挙げます。

### matplotlib-fontja

https://github.com/ciffelia/matplotlib-fontja

Matplotlibを日本語表示に対応させる設定を簡単に行えるようにするライブラリです。既存のライブラリがPython 3.13で動かなくなったため、フォークして作りました。期せずして私が公開したOSSの中で最も多くのスターを獲得しています。

```python
import matplotlib.pyplot as plt
import matplotlib_fontja

plt.plot([1, 2, 3, 4])
plt.xlabel('簡単なグラフ')
plt.show()
```

### nid

https://github.com/ciffelia/nid

Rustで[Nano ID](https://github.com/ai/nanoid)を扱うためのcrateです。Rustの型システムを利用し、文字数や文字種の制約を型に持たせる機能を備えています。ドキュメントも充実させており、気に入っているプロジェクトの一つです。

```rust
use nid::{alphabet::Base62Alphabet, Nanoid};

// Generate a new Nano ID and print it.
let id: Nanoid = Nanoid::new();
println!("{}", id);

// Parse a string into a Nano ID and convert it back to a string.
let id: Nanoid = "abcdefg1234567UVWXYZ_".parse()?;
let s = id.to_string();

// Parse a string into a Nano ID with a different length and alphabet.
let id: Nanoid<9, Base62Alphabet> = "abc123XYZ".parse()?;
```

### wsl-win32-openssh-agent

https://github.com/ciffelia/wsl-win32-openssh-agent

WSLからWin32-OpenSSHのAgentを簡単に利用するためのツールです。Windows security contextに秘密鍵を保存し、WSL内での秘密鍵管理を不要にします。

### leetgen

https://github.com/ciffelia/leetgen

英数字からなる文字列をLeetに変換するウェブアプリです。CTFのフラグをつくるとき便利です。

### uuidgen

https://github.com/ciffelia/uuidgen

ランダムに生成したUUID (v4)を生成するウェブアプリです。ゼロクリックで大量のUUIDを生成できるのが便利です。
